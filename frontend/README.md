## Proje Hakkında

**LastikOtelim**, lastikçiler için geliştirilen modern, sade ve kullanımı kolay bir takip sistemidir.

Bu sistem; müşteri bilgilerini, araç plakalarını, lastik emanet kayıtlarını, depo konumlarını ve etiket yazdırma süreçlerini tek panel üzerinden yönetmeyi amaçlar.

Lastik işletmelerinin manuel takip yükünü azaltır, kayıtların daha düzenli tutulmasını sağlar ve müşteriye ait lastik geçmişini daha kolay yönetilebilir hale getirir.

---

## Temel Özellikler

- Dashboard ekranı
- Müşteri kayıt yönetimi
- Plaka / araç takibi
- Lastik emaneti oluşturma
- Lastik türü, marka, ebat ve adet bilgisi
- Depo konumu takibi
- Fotoğraflı lastik kaydı
- Lastik kaydına özel **LastikCode** oluşturma
- Etiket yazdırma
- Eski / silinmiş kayıtları takip etme
- Abonelik ekranı
- İşletme bilgileri alanı
- Mobil uyumlu responsive arayüz

---

## Kullanım Senaryosu

Lastikçiye gelen bir müşteri için önce müşteri ve plaka bilgisi oluşturulur.

Ardından müşteriye ait lastikler; tür, marka, ebat, adet, fotoğraf ve depo konumu bilgileriyle sisteme kaydedilir.

Her lastik kaydı için özel bir **LastikCode** oluşturulur. Bu kod etiket olarak yazdırılarak lastiklerin fiziksel takibini kolaylaştırır.

Müşteri lastiklerini teslim aldığında kayıt sistemden silinse bile eski kayıtlar üzerinden silinme zamanı ve geçmiş bilgiler takip edilebilir.

## Kurulum

Projeyi local ortamda çalıştırmak için aşağıdaki adımları takip edebilirsin.

### Gereksinimler

Bilgisayarında **Node.js** kurulu olmalıdır.

### 1. Bağımlılıkları yükle

```bash
npm install
```

### 2. Geliştirme sunucusunu başlat

```bash
npm run dev
```

### 3. Tarayıcıda aç

Proje genellikle aşağıdaki adreslerden birinde çalışır:

```bash
http://localhost:3000
```

veya Vite varsayılan portu kullanılıyorsa:

```bash
http://localhost:5173
```
